Погода в Туле 🌤️

Простое и красивое веб-приложение для отображения прогноза погоды в городе Тула на ближайшую неделю. Данные получаются с бесплатного API Open-Meteo.


 ✨ Особенности

- 🔮 Прогноз на 7 дней (максимальная/минимальная температура, скорость ветра)
- 🎨 Современный дизайн с эффектом "стекла" (glassmorphism)
- 📱 Полностью адаптивен для мобильных устройств
- ⚡ Быстрая загрузка и плавные анимации

 🚀 Установка и запуск

Простой способ (рекомендуется)
Просто откройте файл `index.html` в любом современном браузере (Chrome, Firefox, Safari, Edge).

 Локальный сервер (например, с помощью VS Code)
1. Установите расширение **Live Server** в VS Code.
2. Щелкните правой кнопкой мыши по `index.html` → **Open with Live Server**.
